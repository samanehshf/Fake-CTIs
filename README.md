**Fake-CTIs**
## False alarms, real damage: Adversarial attacks using LLM-based models on text-based cyber threat intelligence systems

This study investigates vulnerabilities across the entire CTI pipeline, focusing on how adversarial attacks such as evasion, flooding, and poisoning can disrupt system performance. It shows that adversarially generated fake cybersecurity text can significantly mislead ML and NLP-based classifiers, with very high false positive rates.

All the code and dataset would be available after acceptance of paper "False Alarms, Real Damage: Adversarial Attacks using LLM-based models on
Text-based Cyber Threat Intelligence Systems".
